# 📝 用户名格式说明

## ✅ 用户名格式要求

### 允许的字符
- **英文字母**：a-z, A-Z（大小写均可）
- **数字**：0-9
- **下划线**：_

### 长度限制
- **最少**：3个字符
- **最多**：50个字符

### 格式规则
- 只能包含字母、数字和下划线
- 不能包含空格
- 不能包含特殊符号（如 @, #, $, %, &, *, -, +, 等）
- **不支持中文**

**说明**：如果你看到这个错误，说明你使用的是旧版本的前端代码。请刷新页面重新加载最新代码。

### 错误2：包含中文
```
❌ 错误：尚西西（中文）
✅ 正确：shangxixi（拼音）或 shang_xixi
```

**注意**：`shangxixi` 和 `archer` 这样的纯字母用户名是完全符合格式的！

### 错误3：使用连字符
```
❌ 错误：test-user
✅ 正确：test_user（使用下划线代替）
```

### 错误4：包含空格
```
❌ 错误：test user
✅ 正确：test_user 或 testuser
```

### 错误5：使用邮箱格式
```
❌ 错误：user@example.com
✅ 正确：user_example（用户名和邮箱是分开的字段）
```

### 错误6：太短
```
❌ 错误：ab（只有2个字符）
✅ 正确：abc 或 ab1（至少3个字符）
```

## 📋 其他字段格式要求

### 昵称（Nickname）
- **允许**：任意字符（包括中文）
- **长度**：2-50个字符
- **示例**：张三、Test User、小明123

### 邮箱（Email）
- **格式**：标准邮箱格式
- **示例**：user@example.com、test@qq.com

### 手机号（Phone）
- **格式**：中国大陆手机号
- **规则**：1开头，第二位是3-9，共11位数字
- **示例**：13800138000、15912345678

### 密码（Password）
- **长度**：6-20个字符
- **允许**：任意字符
- **建议**：包含字母、数字和特殊符号

## ❓ 常见问题

### Q1: 为什么不支持中文？
**A**: 为了保证系统兼容性和URL友好性。用户名可能会出现在URL中，中文需要编码，不够简洁。

### Q2: 可以使用邮箱作为用户名吗？
**A**: 不可以。邮箱中的 `@` 和 `.` 不符合用户名格式。但你可以使用邮箱登录（系统支持用户名或邮箱登录）。

### Q3: 用户名可以修改吗？
**A**: 目前不支持修改用户名。但可以修改昵称、邮箱等其他信息。

### Q4: 忘记用户名怎么办？
**A**: 可以使用邮箱登录，或者通过找回密码功能（输入邮箱）来重置密码。

### Q5: 用户名区分大小写吗？
**A**: 是的，`TestUser` 和 `testuser` 是不同的用户名。
