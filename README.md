# 宠物口粮官网

## 项目简介
这是一个专业的宠物口粮官方网站项目，提供宠物食品展示、购买和相关服务功能。

## 项目结构
```
├── demo/           # 演示版本
├── frontend/       # 前端代码
├── doc/           # 项目文档
├── AI_TEST_TEMP/  # AI测试临时文件
└── README.md      # 项目说明
```

## 技术栈
- 前端：HTML5, CSS3, JavaScript
- 框架：WordPress主题开发
- 样式：响应式设计
