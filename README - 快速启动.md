# 🚀 猫砂管理系统 - 快速启动指南

## 一键启动所有服务

### 🎯 最简单的方式

**双击运行：** `启动所有服务.bat`

就这么简单！脚本会自动：
1. ✅ 检查并启动MySQL数据库
2. ✅ 检查Redis服务
3. ✅ 测试数据库连接
4. ✅ 启动后端API服务 (http://localhost:3000)
5. ✅ 启动后台管理界面 (http://localhost:80)
6. ✅ 启动前端网站 (http://localhost:8080)
7. ✅ 自动在浏览器中打开所有页面

---

## 📊 服务访问地址

启动完成后，浏览器会自动打开以下页面：

| 服务 | 访问地址 | 说明 |
|------|----------|------|

## ⚠️ 注意事项

1. **首次使用前确保：**
   - ✅ MySQL服务已安装并运行
   - ✅ Redis服务已安装
   - ✅ 已运行 `npm install` 安装依赖

2. **如果启动失败：**
   - 检查端口是否被占用（3000, 80, 8080）
   - 确认MySQL和Redis服务正常运行
   - 查看服务窗口的错误信息

3. **服务窗口不要关闭：**
   - 每个服务会在独立窗口运行
   - 关闭窗口 = 停止服务
   - 可以在窗口中查看实时日志
